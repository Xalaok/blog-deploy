Windows 11 Moment 4 更新

附大陆开启 Copilot 方法

2023-09-27
阅读时长: 1 分钟
字数: 401 字
XalaokGPT
Windows 11 更新 KB5030310 带来了 Copilot 正式版、WinUI 化的文件资源管理器、任务栏音量混合器、桌面标签等功能改进。用户可隐藏系统托盘中的时间、直接解压多种格式文件、使用 Windows 备份应用程序等。Copilot 需要手动开启，通过下载 Vive Tool 并运行特定代码实现。更新增加了新功能，提升了用户体验，但部分功能仍有改进空间。

在 9/26 微软开放下载了 Windows 11 的又一个更新（KB5030310），更新完这个可选性更新后系统版本号升级至 22621.2361

更新了什么

  1. 最重磅的当属 Copilot 预览版下放正式版 但是众所周知的原因环大陆发行 手动开启方法会在下文介绍

    但是目前 Copilot 能做的事很有限 只是相当于本地 New Bing 而已 而且依托 Edge 后台运行

  2. 文件资源管理器 WinUI 化、加入新详细信息窗格 但是不能通过地址栏拖放文件了

  3. 文件资源管理器标签页加入分离和合并 但是卡的一言难尽 跟浏览器的那种根本没法比

  4. 在任务栏即可使用音量混合器（快捷键 WIN + CTRL + V）

  5. 任务栏图标不合并回归

  6. 切换桌面时会出现桌面标签

  7. 可以隐藏系统托盘中的时间和日期

  8. 可以直接解压 .7z .rar .takes .tar.gz .tar.bz2 .tar.zst .tar.xz .tgz .tbz2 .tzst .tx 压缩文件了 但还不支持加密压缩包

  9. 引入了 Windows 备份应用程序

  10. 设置添加了新的主页

手动开启 Windows Copilot

  1. 在 GitHub 下载 Vive Tool.
  2. 管理员运行终端并 cd 到 Vive Tool 文件夹
  3. 运行以下代码开启
vivetool /enable /id:44774629,44776738,44850061,42105254,41655236
  1. 重启电脑即可在任务栏看到 Copilot 图标 如果没有，可以在设置-个性化-任务栏中开启

如需关闭此功能运行 vivetool /disable /id:44774629,44776738,44850061,42105254,41655236 后重启电脑即可
