绕过 Windows 11 安装时的一些限制

2023-07-10
阅读时长: 1 分钟
字数: 247 字

Windows 11 安装时必需要求有 TPM 2.0 模块而且要联网（22H2 及以后），否则就无法安装系统，这篇文章将会介绍如何绕过这些限制。

绕过 TPM 检查#

  1. 在 Windows 刚开始安装的界面中，按下 Shift + F10 调出命令行，输入 regedit 打开注册表编辑器

  2. 在 HKEY_LOCAL_MACHINE\SYSTEM\Setup 下新增一项 LabConfig，再增加两个值，来禁用安全启动内存检查和 TPM 检查：

  • BypassTPMCheck REG_DWORD 1
  • BypassSecureBoot REG_DWORD 1
  • BypassRAMCheck REG_DWORD 1

或者直接在命令行输入以下代码：

REG ADD HKLM\SYSTEM\Setup\LabConfig -v BypassTPMCheck -t REG_DWORD -d 1
REG ADD HKLM\SYSTEM\Setup\LabConfig -v BypassSecureBoot -t REG_DWORD -d 1
REG ADD HKLM\SYSTEM\Setup\LabConfig -v BypassRAMCheck -t REG_DWORD -d 1

绕过网络连接（22H2）#

按下 Shift + F10，调出命令行界面，输入 oobe\BypassNRO.cmd 回车。系统会自动重启然后重新开始安装并不再强制要求连接网络了
