GitHub 开启 2FA

必须开启二次验证了！

2023-09-29
阅读时长: 1 分钟
字数: 487 字
XalaokGPT
GitHub 从 2023/09/28 开始要求有代码提交的用户开启两步验证（2FA）。用户可以选择身份验证器应用或手机短信验证，但中国号码无法使用手机短信验证。推荐使用 TOTP 应用（如微软 Authenticator）或浏览器扩展（如 Authenticator: 2FA Client）。操作简单：打开 GitHub 设置，启用两步验证，扫描二维码，即可展示动态验证码。这一举措旨在提高 GitHub 账户的安全性。详细信息可查看 GitHub 官方文档。

现在打开 GitHub 会要求有代码提交的用户开启两步验证，否则后面提交不了代码

Because of your contributions on GitHub, two-factor authentication will be required for your account starting Sep 28, 2023. Thank you for helping keep the ecosystem safe! Learn more about our two-factor authentication initiative.
Because of your contributions on GitHub, two-factor authentication will be required for your account starting Oct 6, 2023. Thank you for helping keep the ecosystem safe! Learn more about our two-factor authentication initiative.

GitHub 提供了两种方式：身份验证器应用或手机短信验证

但是从官方文档可知，GitHub 手机短信验证不支持中国号码，所以没有国外号码的话只能考虑 TOTP 应用或浏览器扩展

TOTP应用

以微软的 Authenticator 为例

  1. GitHub 右上角选择【设置】，或者打开 https://github.com/settings/security

  2. 页面拉到底点击 Enable two-factor authentication 打开的页面会显示一个二维码

  3. 打开 Authenticator 应用，从右上角的 + 号添加帐户，选择其他帐户，然后扫一扫二维码

  4. 扫描完成之后帐号就添加到 Authenticator 中了，下次打开切到 GitHub 即可展示动态验证码

浏览器扩展

以开源免费的 Authenticator: 2FA Client 为例

  1. GitHub 右上角选择【设置】，或者打开 https://github.com/settings/security
  2. 页面拉到底点击 Enable two-factor authentication 打开的页面会显示一个二维码
  3. 点击扩展上的扫描按钮，用鼠标选中二维码，即可添加 GitHub帐号

开源地址：https://github.com/Authenticator-Extension/Authenticator
Edge扩展：Authenticator: 2FA Client - Microsoft Edge Addons
Chrome扩展：身份验证器 - Chrome 应用商店
