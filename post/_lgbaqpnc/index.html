初中物理电学概念及公式汇总

2023-06-04
阅读时长: 2 分钟
字数: 521 字
  1. 电荷量 $Q$：物体所带电荷的多少，单位：库伦（$C$）
  2. 电流 $I$：每秒钟内通过导体横截面的电荷量，单位：安培（$A$） $I = \frac Q t$
  3. 电压 $U$：是电荷定向移动形成电流，单位：伏特（$V$）
  4. 电阻 $R$：反映导体对电流阻碍作用，单位：欧姆（$Ω$） $1Ω = 1 \frac V A$
  5. 电阻率 $ρ$：表示各种物质电阻特性，单位：欧·米（$Ω·m$）
  6. 欧姆定律：$I = \frac U R$
  7. 电阻定律：$R = ρ \frac l S$
  8. 串联电路（分压）：
    • $I = I_{1} = I_{2}$
    • $U = U_{1} + U_{2}$
    • $R = R_{1} + R_{2}$
    • $\frac{U_{1}}{R_{1}} = \frac{U_{2}}{R_{2}} = \frac{U}{R}$（由欧姆定律）
    • $\frac{U_{1}}{U_{2}} = \frac{R_{1}}{R_{2}}$（由欧姆定律）
    • $U_{1} : U_{2} : U_{3} : \cdots : U_{n} : I = R_{1} : R_{2} : R_{3} : \cdots : R_{n} : R$ （推广）
  9. 并联电路（分流）：
    • $I = I_{1} + I_{2}$
    • $U = U_{1} = U_{2}$
    • $\frac{1}{R} = \frac{1}{R_{1}} + \frac{1}{R_{2}}$
    • $I_{1} : I_{2} : I_{3} : \cdots : I_{n} : I = \frac{1}{R_{1}} : \frac{1}{R_{2}} : \frac{1}{R_{3}} : \cdots : \frac{1}{R_{n}} : \frac{1}{R}$（推广）
  10. 电功 $W$：电流所做的功，单位：焦耳（$J$ ） $W = UIt$
  • $W = QU$ （由 $I = \frac Q t$）
  • 纯电阻电路：$W = \frac {U^{2}} {R} t = I^{2} R t$ （由欧姆定律）
  1. 电功率 $P$：单位时间内电流做的功，表示电流做功快慢，单位：瓦特（$W$），千瓦（$kW$） $W = \frac P t = UI$
  • 纯电阻电路：$P = \frac {U^{2}} {R} = I^{2} R$ （由欧姆定律）
  1. 额定电压：用电器正常工作时的电压。
  2. 额定功率：用电器在额定电压下消耗的电功率
